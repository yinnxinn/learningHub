## 推荐理由
- 中文社区口碑教程，覆盖变量、函数、面向对象、并发等核心概念。
- 每一章都配有习题与参考答案，方便自测与巩固。
- 仓库提供幻灯片与源码，可直接拉取在本地运行。

## 学习建议
1. 按章节顺序完成练习，记录错题与难点。
2. 针对不熟悉的知识点，结合官方文档或实战项目再复习一遍。
3. 完成全部 50 题后，整理笔记或编写总结文章，加深理解。
