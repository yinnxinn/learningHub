## 推荐理由
- 由官方维护，示例紧跟最新 API。
- 以对比方式解释 Python 原生列表与 ndarray 的差异。
- 涵盖广播机制、索引切片、聚合运算，是理解后续数值计算的关键。

## 学习路线
1. 先通读页面，确保理解 ndarray 的构建方式与基本属性。
2. 针对广播与向量化章节，尝试复现示例并修改维度验证规则。
3. 结合 Kaggle 或 LeetCode 数值类题目，用 ndarray 改写以体会性能差异。
