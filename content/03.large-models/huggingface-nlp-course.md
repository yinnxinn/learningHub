## 推荐理由
- 课程结合文档、视频与 Colab Notebook，动手体验最主流的 LLM 工具链。
- 涵盖分词、微调、推理、部署、评测等模块，兼顾理论与实践。
- 与 `transformers`、`datasets`、`evaluate` 等库紧密结合，贴近生产环境。

## 学习建议
1. 按章节顺序完成 Notebook，确保理解每一步的 API 用法。
2. 尝试在免费 GPU（如 Colab/Kaggle）上复现微调流程。
3. 使用课程提供的评估指标，对比不同模型或参数组合的效果。
