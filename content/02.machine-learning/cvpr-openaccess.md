## 推荐理由
- 提供历年 CVPR、ICCV、ECCV 论文与补充材料的统一入口。
- 支持按年份、分会场筛选，也可直接搜索关键词。
- 包含视频报告与海报，有助于快速了解论文亮点。

## 阅读建议
1. 结合研究方向，筛选 Workshop 或 Oral Session 的论文优先阅读。
2. 关注补充材料（Supp），了解实验细节与额外实验。
3. 与 GitHub 搜索结合，查找作者发布的代码或复现项目。
