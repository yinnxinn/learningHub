## 推荐理由
- 面向初学者，讲解线性回归、逻辑回归、SVM、聚类等常见算法。
- 每周配套编程作业与测验，巩固理论理解。
- 课程社区活跃，可与全球学习者讨论，共享笔记与工具。

## 学习建议
1. 严格跟随课程安排，按周完成作业，避免一次性“刷完”。
2. 将作业迁移到 Python + scikit-learn 实现，熟悉现代工具链。
3. 完成课程后复盘总结，梳理每个算法的适用场景与优缺点。
